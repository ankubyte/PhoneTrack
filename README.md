# 📱 PhoneInfo API

**PhoneInfo** is a Python-based tool that fetches detailed information about phone numbers using an API. It provides valuable insights like the carrier, location, number type, and more. This can be especially useful for validating or enriching phone number data.

## 🛠️ Features

- 🔍 Fetches phone number details such as:
  - 📡 Carrier Information
  - 🌍 Location Data (Country, State, City)
  - 📞 Phone Number Type (Mobile, Landline)
  - ✔️ Validity check

## 📝 Requirements

To run this code, you'll need:
- 🐍 Python 3.x
- 📦 Requests library (install via `pip install requests`)

## ⚙️ Installation

Follow the steps below to get the project up and running on your local machine:

1. **Clone this repository**:

   ```bash
   git clone https://github.com/Ankuucode/PhoneInfo.git
   cd Phonetrack
   pip install -r requirements.txt
   ```
2. **Run command**
   ```bash
   python phoneinfo.py
   ```
